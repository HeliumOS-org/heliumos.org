Roadmap

Current progress and future plans for HeliumOS

  • June 2024

    HeliumOS v9-Alpha available

    Based on CentOS Stream 9, this release establishes HeliumOS as an atomic Linux distribution using bootc for immutability.

    {% include "www/snippets/timeline_point_done.html" %}
  • {% include "www/snippets/timeline_point_done.html" %}
    July 2024

    HeliumOS rebase on AlmaLinux

    Among other changes, HeliumOS rebases on AlmaLinux 9.4. This change stabilizes the Linux kernel and other software, allowing for additional features.

  • {% include "www/snippets/timeline_point_not_done.html" %}
    August 2024

    Exciting improvements

    This update to the HeliumOS 9 Alpha adds the "System" Images app. A graphical screen is displayed during boot, including a nice prompt for disk encryption. Many bugs such as the slow session exit bug are patched.
